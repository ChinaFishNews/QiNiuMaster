<H1>七牛云存储批量管理工具</H1>
<pre>QiNiuMaster, a batch tool to help you manage your files on the QiNiu Cloud Storage easily.</pre>
<H3>   </H3>
<pre>
版本1.0 (2015.2.21)
七牛云存储批量管理工具，可以方便地对七牛云存储上的文件进行大批量的下载和删除等操作。
GitHub开源地址：http://github.com/MoonLord-LM/QiNiuMaster
